#  The Last Minute Life Saver

An AI-powered productivity companion designed to help users beat deadlines, plan tasks efficiently, and stay productive.

##  Features

-  Task Management
-  Calendar for scheduling tasks
-  AI Assistant for productivity help
-  Deadline Risk Prediction
-  Progress Tracking
-  Streak Counter
-  Achievement Badges
-  Analytics Dashboard
-  Light/Dark Mode

##  Tech Stack

### Frontend
- React.js
- Vite
- React Router
- Chart.js
- CSS

### Backend
- Python
- FastAPI
- Axios

##  Screens

- Dashboard
- Calendar
- AI Assistant
- Analytics
- Settings

##  Problem Statement

Many students and professionals procrastinate and struggle with managing deadlines. The Last Minute Life Saver helps users organize tasks, track productivity, and receive AI-powered assistance to complete work on time.

##  Installation

### Clone the repository

```bash
git clone https://github.com/Anyaverse/The-Last-Minute-File-Saver.git
```

### Install dependencies

```bash
npm install
```

### Run the frontend

```bash
npm run dev
```

### Run the backend

```bash
python main.py
```
